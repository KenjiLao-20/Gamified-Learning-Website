# 👥 Zetatech - User Management System

A role-based authentication system for Zetatech University LMS with login, registration, and password recovery.

<br>

## 🔐 Features

- User Login with email + password
- Account Registration (unique email check)
- Password Recovery / Change
- Dark Mode Toggle (saved locally)
- Role-Based Access Control

<br>

## 👑 User Roles & Permissions

| Role | Access Level |
|------|-------------|
| **Student** | Limited access (no course creation or user management) |
| **Instructor** | Can create courses, cannot manage users |
| **Admin** | Full access to all features |
